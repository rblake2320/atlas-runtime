# Open Gaps

The wrapper is stronger than the first baseline, but these gaps remain open and are intentionally not hidden.

- `24h+` autonomy is not yet proven.
- Learning-proof deltas are still zero in the wrapper proof surface.
- External-value wins are still zero in the wrapper proof surface.
- External provider adapters exist, but provider-level execution has not been verified with live credentials.
- The multi-agent loop is deterministic and owner-aware, but it is not yet a general planner/delegator.
- Public packaging beyond GitHub is still missing.
